4.【消消乐游戏】
【消消乐游戏】游戏规则：输入一个只包含英文字母的字符串，字符串中的两个字母如果相邻且相同，就可以消除。

在字符串上反复执行消除的动作，直到无法继续消除为止，此时游戏结束。

输出最终得到的字符串长度。

思想：

替换，遍历数组，把当前字符重复两次组成字串，若在输入字符中,则换成空格，则把i-1，否则为i+1

```java
public class DelLetterGame {
    public int fun(String input) {
        int i = 0;
        while (i < input.length()) {
            String repeat = input.substring(i, i + 1).concat(input.substring(i, i + 1));
            if (input.contains(repeat)) {
                input = input.replace(repeat, "");
                i = Math.max(0, i - 1);
            } else {
                i++;
            }
        }
        return input.length();
    }

    public static void main(String[] args) {
        try (Scanner sc = new Scanner(System.in)) {
            while (sc.hasNext()) {
                DelLetterGame dlg = new DelLetterGame();
                System.out.println(dlg.fun(sc.nextLine()));
            }
        }
    }
}
```

class DelLetterGame:
    def fun(self, nums: str) -> int:
        i = 0
        while i < len(nums):
            if nums[i] * 2 in nums:
                nums = nums.replace(nums[i] * 2, "")
                i = max(0, i - 1)
            else:
                i += 1
        return len(nums)
